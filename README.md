# 🎁 Amigo Secreto Cyberpunk

Um sistema web simples e estilizado para organizar sorteios de amigo secreto com uma estética cyberpunk 🌆⚡

## 🚀 Funcionalidades

- ✅ Adicionar participantes dinamicamente
- 🔢 Validação com mínimo de 4 participantes
- 🔀 Sorteio automático sem repetição (ninguém tira a si mesmo)
- 🔒 Consulta individual (cada participante vê apenas o seu resultado)
- 💾 Persistência com LocalStorage (dados salvos no navegador)
- 📧 Compartilhamento rápido via:
  - WhatsApp
  - Email
- 🎨 Interface com estilo cyberpunk (cores neon e tema escuro)

## 🧠 Como Funciona

1. Os participantes são adicionados manualmente
2. Ao clicar em **Sortear**, o sistema:
   - Embaralha os nomes
   - Garante que ninguém tire a si mesmo
3. O resultado é salvo no navegador (LocalStorage)
4. Cada pessoa pode consultar seu amigo secreto digitando seu nome

## ⚠️ Limitações

- Não há autenticação (qualquer pessoa pode consultar qualquer nome)
- Dados ficam apenas no navegador (não há banco de dados)
- Compartilhamento não envia automaticamente (apenas abre apps)
